# ⚠️ CONFIGURAÇÃO DE ENCODING - SEMPRE VERIFICAR

## Configurações Git Obrigatórias

```bash
git config --global core.quotepath false
git config --global i18n.commitEncoding utf-8
git config --global i18n.logOutputEncoding utf-8
git config --global core.editor "notepad"
```

## Antes de Fazer Commit

1. **SEMPRE** usar `chcp 65001` no PowerShell antes de commits
2. **SEMPRE** verificar encoding dos arquivos antes de commit
3. **SEMPRE** usar arquivo temporário para mensagens de commit com acentos

## Exemplo de Commit Seguro

```bash
# 1. Mudar encoding do terminal
chcp 65001

# 2. Criar arquivo temporário com mensagem
echo "fix: corrige aspas nao escapadas no ESLint" > commit_msg.txt

# 3. Fazer commit usando arquivo
git commit -F commit_msg.txt

# 4. Limpar arquivo temporário
del commit_msg.txt
```

## Verificar Encoding de Arquivo

```bash
# PowerShell
Get-Content arquivo.txt -Encoding UTF8 | Out-File -Encoding UTF8 arquivo.txt
```

## ⚠️ REGRA DE OURO

**NUNCA fazer commit com acentos diretamente na linha de comando no Windows PowerShell sem configurar UTF-8 primeiro!**
